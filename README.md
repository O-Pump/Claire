## Claire 可能是最聪明的温度转换器。

Claire是一款安卓平台上的温度检测工具。在拥有温度传感器的手机设备上，Claire会检测出实时的温度信息，并提供摄氏和华氏两种制度的显示。Claire同时是一个摄氏华氏温度转换器。相对于安卓平台上其他的温度转换工具，Claire支持强大的手势操作，包括上下滑动调整温度，摇一摇重置等。
